Express Bootstrap

[![Build Status][1]][2] [![Dependency Status][3]][4] [![Development Dependency Status][5]][6]


## Set up development environment
```
$ npm install
$ bower install
$ gulp compile-scss
```

## Run app
```
$ npm start
```

## Run app in watch mode
```
$ gulp watch
```

## Run app as production
```
$ gulp compile-scss
$ gulp minify
$ NODE_ENV=production npm start
```

## Run Tests
```
$ npm test
```
